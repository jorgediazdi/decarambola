# Apps

Contenedor de las tres aplicaciones funcionales:

- `jugador/` — estructura destino; entrada actual del jugador: **`../jugador/index.html`** (raíz).
- `club/` — portal: **`../club/index.html`**; sala migrada: **`club/sala/`**; organizador/torneos: **`club/organizador/`**.
- `superadmin/` — backoffice (pendiente de pantallas; hoy `pqrs_admin.html` en raíz).

**Mapa de navegación interna:** `apps/index.html` (no sustituye al `index.html` principal).

**Roadmap completo:** `docs/ROADMAP_APPS.md`.

Regla: no borrar HTML de la raíz hasta migrar enlaces y validar deploy.
