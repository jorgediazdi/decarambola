# Mejoras sugeridas — DeCarambola

Documento para **priorizar la evolución del producto**. Se alimenta de:

- **Informe de uso del Sensei** (temas más consultados, consultas sin buena respuesta, sugerencias implícitas).
- **PQRS** (peticiones, quejas, reclamos y **sugerencias** que llegan por Joe).

El admin de la plataforma y el agente (IA/Cursor) pueden usar esta lista para decidir qué mejorar primero: contenido del Sensei, flujos, respuestas tipo para PQRS recurrentes, etc.

---

## Cómo usar este documento

- Añade ítems en las secciones que correspondan (Sensei, PQRS, general).
- Opcional: marca **Prioridad** (Alta / Media / Baja) y **Origen** (Sensei / PQRS / interno).
- Revisa periódicamente y mueve lo implementado a "Hecho" o bórralo.
